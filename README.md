## プロジェクト概要

ORTHRUS（オルトラス）の公式ランディングページ。  
小平市および多摩地域の中小企業経営者（従業員10〜100名）をターゲットに、  
無料相談予約・問い合わせを促進するコンバージョン特化型LP。

## 機能一覧

- **固定ヘッダー** — スクロール追従 + ナビゲーション
- **モバイルメニュー** — ハンバーガーメニュー対応
- **スクロールアニメーション** — Intersection Observer による fade-in
- **FAQアコーディオン** — 開閉アニメーション
- **カウンターアニメーション** — 実績数値のインクリメント表示
- **フォームバリデーション** — 必須項目チェック + 送信成功メッセージ
- **フローティングCTA** — ヒーロー通過後に表示
- **アクティブナビ** — スクロール位置に応じたナビゲーションハイライト
- **完全レスポンシブ** — モバイル / タブレット / PC対応

## 画像の配置方法

`img/` フォルダに以下のファイルを配置してください。

| ファイル名 | 用途 |
|-----------|------|
| `img/logo.png` | ヘッダー・ヒーロー・ブランドセクション・フッターで使用 |

ロゴ画像が存在しない場合、`onerror` 属性により非表示になります。
